# 📊 Projeto com Neo4j e Cypher — Análise de Tweets e Hashtags

Este projeto demonstra a criação de um grafo no **Neo4j**, com dados de tweets, usuários e hashtags coletados de um arquivo `.json`. O objetivo é exercitar os conceitos de banco de dados NoSQL com foco em grafos, usando a linguagem **Cypher** para criação de nós, relacionamentos e consultas.

## 🚀 Tecnologias Utilizadas

- [Neo4j](https://neo4j.com/)
- Cypher Query Language
- APOC (biblioteca de procedimentos auxiliares)
- VS Code com extensão **Cypher Query Language** de Jake Boone
- JSON como fonte de dados

## 📁 Estrutura do Projeto
	**👨🏻‍💻 Commands
--01_create_nodes.cypher (Criação de nós)
--02_create_relationships.cypher (Criação de relacionamentos)
--03_basic_queries.cypher (Comando básico pra grafo)
--04_hashtag_principal (Comando para achar hashtag mais comum)
